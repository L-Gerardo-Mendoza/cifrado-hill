## 6. Pasos rápidos para Git y los commits (lo que me preguntabas de “cómo actualizo y subo”)

Suponiendo que ya creaste el repo en GitHub:

```bash
# 1. Ver qué cambió
git status

# 2. Agregar los cambios al "stage"
git add .

# 3. Crear un commit con mensaje
git commit -m "Mensaje del cambio, por ejemplo: agregar README con explicación"

# 4. Subir a GitHub
git push
